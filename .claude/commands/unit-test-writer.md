## Step 3 — Generate tests

Rules:
- Use **MSTest** and **Moq** only — no xUnit, NUnit, FluentAssertions, or Shouldly
- Mock all constructor-injected dependencies
- Cover: success path, each validation branch, each error case
- Test behavior, not implementation details
- Do not test private methods directly
- Follow the naming and structure of `GeneratePresignedUrlsOperationTests`

Test method naming:
```
<MethodName>_When<Condition>_Should<ExpectedOutcome>
```

Test structure:
```csharp
[TestClass]
public class <ClassName>Tests
{
    [TestMethod]
    public async Task <MethodName>_When<Condition>_Should<ExpectedOutcome>()
    {
        // Arrange

        // Act

        // Assert
    }
}
```

Async exception assertion:
```csharp
await Assert.ThrowsExceptionAsync<SomeException>(async () =>
    await sut.<MethodName>(input, cancellationToken));
```

Assertions: use `Assert.AreEqual`, `Assert.IsNotNull`, `Assert.IsTrue`, `Assert.IsFalse` only.

Verify only calls that matter to the behavior under test:
```csharp
mock.Verify(x => x.MethodAsync(It.IsAny<Input>(), It.IsAny<CancellationToken>()), Times.Once);
```
